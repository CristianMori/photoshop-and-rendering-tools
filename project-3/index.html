Explaination of results for Best Example

Poisson blending works by putting contraints on each individual pixel based off the pixel values of its neighbors. By getting the neighboring values from either the source or target image, the resulting belnded image is a smooth and naturally existing source image within the target image. To make the blending more natural, I focused on the coloring of the target and source image. The more different the colors were, the better the poisson blending was.
